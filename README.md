# 🚀 [My Portfolio]

> My first frontend project: a landing page built with pure HTML5 and CSS3.

This repository tracks my early journey in frontend development. It demonstrates my understanding of HTML structure using semantic tags, CSS styling, and media integration without using frameworks or build tools.

## 🌐 Live Demo
Check out the live site here: **[View Demo](https://warden-alt.github.io/my-portfolio/)**

## 🛠 Tech Stack
- **HTML5** – Semantic structure (`header`, `section`, etc.).
- **CSS3** – Styling, positioning, and layout.
- **Git & GitHub** – Version control and deployment via GitHub Pages.
- **Media Optimization** – Lightweight background video (MP4).

## ✨ Key Features
- **Semantic HTML:** Proper use of `header` and `section` tags for better structure.
- **Performance:** Optimized background video for fast loading.
- **Cross-Browser Support:** Tested on Chrome, Firefox, Safari, and Edge.
- **Custom Styling:** Unique design created entirely with CSS.
